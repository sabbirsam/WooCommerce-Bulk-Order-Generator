== Installation ==

1. Upload wc-bulk-order-generator to /wp-content/plugins/
2. Activate through WordPress plugins menu
3. Go to WooCommerce > Bulk Generator
4. Configure generation settings
5. Start generating orders or products


== Frequently Asked Questions ==

= How many orders can I generate? =
The plugin supports generating up to 1,000,000 orders. We recommend starting with smaller batches to test your server capacity.

= Will this affect my live orders? =
No. Generated orders are clearly marked as test data and can be safely deleted without affecting real orders.

= What about server performance? =
The plugin uses optimized batch processing and caching to minimize server load. You can adjust batch sizes based on your server capacity.
